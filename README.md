# Sorting Algorithms

## Requirements
1. Array of sizes n= 10, 100, 500, 5000 and 25000 items with randomly generated integers between 0 and 2n, n = size of the array
2. Following sort operations.
    1. Bubble sort
    2. Insertion sort
    3. Merge-sort
    4. Quicksort
    5. Heap-sort
    6. Counting sort
    7. Radix-sort
3. Testing the sort operations with std::chrono
